# Changelog

## 1.1.0 (2026-01-17)
- [Added] Central Configuration Management via `ands.config.yaml`
- [Added] `ands config` CLI for configuration lifecycle management
- [Added] Schema Versioning Logic supporting ANDS 1.0 and 1.1
- [Added] `ands migrate` CLI for automated declaration upgrades
- [Added] Environment (E) axis support in ANDS 1.1
- [Improved] Refactored `ands-scan` to respect global configuration defaults

- Initial public release
- ANDS five-axis code definition
- `/.well-known/ands.json` declaration schema
- Chain-of-Work schema
- Reference scanner and validator tooling
